# 求职简历助手

这是一个可直接在浏览器使用的前端求职工具站。当前版本不需要后端、不需要数据库、不需要 API Key，所有输入和生成记录只保存在用户当前浏览器的本地存储里。

## 已可用功能

- 简历评分：根据完整度、岗位关键词、量化结果和表达结构生成本地诊断。
- 岗位匹配：粘贴岗位 JD 后输出匹配度、已匹配/待加强/缺失关键词。
- 自我评价：根据目标岗位和简历生成多版自我评价。
- 项目经历：把项目素材改写成背景、职责、动作、结果结构。
- 面试题：根据简历和岗位生成面试问题与答题方向。
- 模板库：可载入模板、下载模板文本。
- 进度条：根据五个工具的完成状态自动更新。
- 最近记录：自动保存本地生成历史。
- 报告操作：支持复制报告、下载文本报告、清空输入和清空结果。

## 暂不开放

付费能力暂不开放。页面里只保留“后续能力计划”，不会进入支付流程。
