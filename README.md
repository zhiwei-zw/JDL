# 求职简历助手

一个纯前端求职工具站 MVP，适合先用免费静态托管验证流量和需求。

## 功能

- 简历评分：根据完整度、关键词、项目经历和量化结果生成体检报告。
- 岗位匹配：对比简历和 JD，输出已匹配关键词、缺失关键词和补强建议。
- 自我评价：根据目标岗位、年限和经历生成一段可改写的自我评价。
- 项目经历：按背景、职责、方法、难点、结果优化项目描述。
- 面试题：根据岗位 JD 和简历亮点生成面试准备清单。

## 使用方式

直接打开：

```text
E:\career-tools-web\index.html
```

当前版本不需要服务器、不需要数据库、不需要 AI Token，用户输入内容只在浏览器本地处理。

## 部署建议

可以免费部署到：

- Cloudflare Pages
- Vercel
- GitHub Pages

第一版建议先免费运行，观察搜索流量和用户使用情况。后续可以增加：

- 精品简历模板下载
- PDF 导出
- AI 深度润色
- 会员或单次付费
