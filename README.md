# 求职简历助手

面向应届生、转行求职和职场跳槽用户的简历优化工具站。

当前版本已替换为新版「专注工作台」界面，包含：

- 简历评分工作台
- 岗位匹配入口
- 自我评价、项目经历、面试题导航
- 简历体检报告
- 关键词匹配和优化行动清单
- 求职准备进度
- 模板库和会员升级入口

## 部署说明

本仓库根目录是已经打包好的静态网页，可直接用于 GitHub Pages。

保留 `.nojekyll` 是为了让 GitHub Pages 正常加载 `assets/` 下的构建资源。
